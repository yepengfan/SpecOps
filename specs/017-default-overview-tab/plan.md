## Key Design Decisions

### 1. Project Card Link Target (FR-001)

Change the `href` in the project card `<Link>` from `/project/${project.id}/${activePhase}` to `/project/${project.id}/overview`. This is a one-line change. The `getActivePhase` import can be removed if it becomes unused after the change.

**Files**: `components/ui/project-card.tsx`

### 2. No Changes to Other Navigation Paths (FR-002)

The redirect in `app/project/[id]/page.tsx` already points to overview. Phase tab navigation in `phase-nav.tsx` is unchanged. Bookmarked/direct URLs continue to work. Only the project card entry point is modified.
