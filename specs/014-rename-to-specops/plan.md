## Key Design Decision: IndexedDB Database Name

**Decision**: Rename the Dexie database from `"sdd-cockpit"` to `"spec-ops"`. Accept data loss.

**Rationale**: The application is only running locally in development with no production users. A clean rename of all identifiers (including the database name) is preferred over carrying a legacy name internally. Existing local data in the old "sdd-cockpit" database becomes inaccessible — users simply recreate projects as needed.
